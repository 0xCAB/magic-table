greg.ross.FisheyeTable(tableModel, x, y, width, height, tableTitle, targetElement)
This is the library's core class.
Author: Greg Ross.
Parameters:
tableModel
contains the data and meta data for rendering the table
x
the x-axis position of the table
y
the y-axis position of the table
width
the table's width
height
the table's height
tableTitle
the title that appears above the table
targetElement
the element in which the table html will be nested. If this is ommitted the table's position will be absolute, otherwise it will be relative to the target element.
